# ecommerce project

## General description
* Ecommerce backend project - microservices
	
## Technologies
Project is created with:
* Spring Boot
* Maven
* Spring Data Jpa and Hibernate
* Spring Security JWT
* Spring Cloud
* Eureka
* Zipkin 
* Resilience4J
* MySql
* Docker
